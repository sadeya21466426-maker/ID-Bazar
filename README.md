Create a modern mobile-friendly website called ID Bazar.

This website will be a digital marketplace where users can buy and sell online IDs and manage their wallet.

Main Features

1. User Authentication

- Login and Register system
- Email / Username and Password
- Forgot Password option
- Secure authentication

2. User Dashboard
   After login users will see a dashboard with menu options:

- Home
- Buy ID
- Sell ID
- Wallet / Balance
- Withdraw
- Transaction History
- Submit Screenshot
- Support / Admin Chat
- Account Settings

3. Buy & Sell System
   Users can:

- Post IDs for sale
- View available IDs
- Purchase IDs
- Contact seller

4. Wallet System
   Each user has a wallet showing:

- Current Balance
- Deposit
- Withdraw
- Transaction History

5. Withdraw System
   Users can withdraw money using:

- Bkash
- Nagad

Withdraw page fields:

- Withdraw Number
- Select Method
- Select Amount
- Request Withdraw Button

Show withdraw history table with:

- Date
- Number
- Method
- Amount
- Status

6. Screenshot Submission
   Users can submit proof screenshots to verify tasks or transactions.

7. Admin Panel
   Only admin can access it.

Admin can:

- Manage users
- Add or remove balance
- Approve withdraw requests
- Manage buy/sell ID listings
- View submitted screenshots
- Manage payments

8. Design

- Mobile responsive
- Clean modern UI
- Blue and white theme
- Sidebar mobile menu
- Dashboard style similar to earning platforms.

Website name: ID Bazar
